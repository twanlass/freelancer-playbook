static-gulp
===========

Simple front-end build stack for Statamic

#Setup

## Update node

1. ``$ brew update``
2. ``$ brew doctor``
3. ``$ brew upgrade node``

## Clone repo

`$ git clone git@github.com:we-are-altitude/static-gulp.git`

## Install dependencies

`$ npm install`

## Launch gulp

`$ gulp`

## Build site

`$ gulp build`

## Build site for production

`$ gulp build --production`
